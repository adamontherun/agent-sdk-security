A hands-on course

Agent SDK Security

The Claude Agent SDK will run whatever a task calls for: shell commands, file edits, network requests. That's the whole point of an agent, and it's also the whole problem — an agent's behavior is shaped by everything it reads, including content an attacker controls. This course is about the layer between "the agent decided to do this" and "the agent was able to do this": permission rules, OS-level sandboxing, container and microVM isolation, and a network boundary the agent can't see past. Every proxy, container, and IaC template in this book is one you build and actually run, not one described in the abstract.

Each chapter pairs prose with a runnable example and a challenge with failing tests you make pass. Chapters 10 and 15 build a real hardened Docker container and a real Squid egress proxy on your own machine; chapters 18 and 19 write complete Terraform for Amazon ECS and AWS Lambda MicroVMs that you validate with the real CLI tooling.
